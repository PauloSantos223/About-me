My Blog: The Details Tag

Simple Example:

Click Me

Hi! This is a message that appeared only because you clicked where the "Click Me" text is located.


The details tag in HTML5 specifies additional details that a user can open or close on demand. This can be used in a number of ways from specifying important information a user may not know about, to just displaying an important message to the user before reading about something. It is often used to create an interactive widget that is accessible to users. It can also be styled with css and supports the global attributes of HTML.

As you can see from the second example, you can add a number of different styles to your details tag although this particular example being more advanced. The normal state is decorated with a different type of border and border shadow along with a hover state that turns the whole text into a color gradient. While selecting the main details section and highlighting the hover position can be done within css as such: exampleid:hover. The details tag needs to be immediately followed by a summary tag in which the text "Click Me" is represented, and after a paragraph tag to write the detail of said summary.
